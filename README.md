# oasis-infobyte-data-science-task-1
---

Iris Flower Classification
---
This project was developed during my Data Science internship at Oasis Infobyte. The goal of this project was to apply machine learning classification techniques to predict the species of Iris flowers based on measurable physical attributes.

---

📌 Project Objective
To build a robust machine learning classification model that accurately predicts the species of an Iris flower (Iris-setosa, Iris-versicolor, or Iris-virginica) using four key physical features:

Sepal Length

Sepal Width

Petal Length

Petal Width

---
🛠️ Key Steps Followed
Exploratory Data Analysis (EDA): Performed comprehensive analysis to visualize the relationship between species and physical traits using scatter plots, pair plots, and heatmaps to identify strong correlations.

Data Preprocessing: Cleaned the dataset and split the data into training and testing sets (using a standard ratio) to ensure the model could generalize well to new, unseen data.

Model Building: Implemented the Random Forest Classifier, a highly effective algorithm for multi-class classification, to learn the underlying patterns in the measurements.

Performance Evaluation: Validated the model using a Confusion Matrix and classification reports to ensure high precision, recall, and overall accuracy across all three target species.
